# Network Security Labs

Educational network security proof-of-concepts for learning how common network-level attacks and detections work in controlled lab environments.

This repository focuses on *understanding* network threats by implementing them in a minimal, transparent way.

## Purpose & scope
These tools are intended for:
- learning network security fundamentals
- lab environments
- defensive awareness (understanding attacker techniques)

They are **not** designed for real-world misuse or unauthorized environments.

## Tech
- Python
- Scapy
- Linux networking primitives

## Notes
The implementations are intentionally simple and readable to make the underlying concepts easy to follow.
